# 🔐 مِرآة — لوحة الأدمن المستقلة

> موقع منفصل تماماً لإدارة كودات لعبة مِرآة.
> يتصل بنفس قاعدة بيانات Upstash التي تستخدمها اللعبة.

---

## نبذة

لوحة إدارة خاصة لإنشاء كودات لعبة مِرآة، تتبع اللاعبين، وعرض الإحصائيات والإيرادات. مفصولة عن موقع اللعبة للحماية القصوى.

> 🔒 **مستودع خاص:** هذا المستودع يجب أن يكون Private دائماً.

## خطوات النشر على Netlify

### 1️⃣ ربط Netlify بالمستودع
- Add new site → Import from GitHub → اختاري `miraah-admin`
- Publish directory: `public`

### 2️⃣ متغيرات البيئة المطلوبة

| المتغير | الوصف |
|---|---|
| `UPSTASH_REDIS_REST_URL` | رابط قاعدة بيانات Upstash |
| `UPSTASH_REDIS_REST_TOKEN` | توكن المصادقة |

> 💡 **مهم:** يجب أن تكون القيم **متطابقة تماماً** مع المتغيرات المستخدمة في موقع اللعبة، ليتمكن الاثنان من مشاركة نفس قاعدة البيانات.

> 🔒 القيم الفعلية محفوظة عند المالكة فقط.

### 3️⃣ النطاق المعتمد
```
miraah-admin.netlify.app
```
