## نبذة

لعبة سينمائية مسرحية ذكية تفاعلية، عشرة فصول تصاعدية، يُخرجها بوتٌ هو في جوهره **مخرجٌ سينمائيٌ مسرحيٌ مخضرم**، يكتب كلّ سؤالٍ كمشهدٍ، ويعلّق على كلّ اختيارٍ كأنه صوتٌ من خلف الكاميرا.

**الخلطة الفريدة:** رعبٌ نفسيٌ اجتماعيٌ بوليسيٌ كوميديٌ لايت، بفصحى مسرحية بلمسات شاعرية.

**النماذج الأدبية المُستلهَم منها:** توفيق الحكيم، نجيب محفوظ، سعدالله ونوس، الطيب صالح، جبران خليل جبران، محمد الماغوط، غادة السمّان، علي أحمد باكثير.

## الميزات السينمائية الجديدة

| الميزة | الوصف |
|---|---|
| **شاشة الإهداء** | تظهر لوالدكِ عند دخول الموقع، تُخفى تلقائياً بعد 7 ثوانٍ |
| **ستارة المسرح** | ترتفع بصرياً قبل كل فصل مع عنوان الفصل |
| **إرشاد المخرج** | يظهر قبل كل مشهد (مثل: *"تتحرك الكاميرا ببطء…"*) |
| **إشارة صوتية** | توجيه صوتي مكتوب (مثل: *"♪ صوت ماء بعيد"*) |
| **صوت من خلف الكاميرا** | تعليق المخرج على المشهد السابق |
| **قفلة الفصل** | مونولوج ختامي بفصحى مسرحية + جملة إغلاق شاعرية |
| **شريط الفيلم** | مؤشر تقدم بشكل شريط سينمائي مكون من 100 إطار |
